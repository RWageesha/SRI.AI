මාතෘකාව: Cloud Computing

Cloud Computing යනු internet හරහා servers, storage, databases සහ software සේවා ලෙස ලබා දීමයි. Local infrastructure අවශ්‍යතාවය අඩු කිරීමට මෙය උපකාරී වේ.

Cloud සේවාවන්ගේ ප්‍රධාන වාසි ලෙස scalability, flexibility සහ pay-as-you-use model එක සඳහන් කළ හැක. අවශ්‍යතාවය අනුව resources වැඩි/අඩු කළ හැක.

IaaS, PaaS, SaaS වැනි cloud service models පවතී. Data security, backup සහ access control නිවැරදිව සැලසුම් කිරීම cloud භාවිතයේදී වැදගත් වේ.

අමතර විස්තර (විස්තීරණය):
- ප්‍රධාන කරුණු: Cloud Computing සඳහා architecture components, data flow, සහ reliability considerations එකට බලන විට system understanding වැඩි වේ.
- ප්‍රායෝගික භාවිතය: setup -> monitor -> backup -> recovery යන workflow එකක් අනුගමනය කිරීමෙන් stable operations පවත්වා ගත හැක.
- සම්බන්ධ උපමාතෘකා: configuration management, observability, access control, සහ patch/update strategy.
- හොඳ පුරුදු: documentation update, periodic health checks, සහ security hardening steps ක්‍රමානුකූලව යෙදීම.
