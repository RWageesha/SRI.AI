මාතෘකාව: API

API (Application Programming Interface) යනු software පද්ධති දෙකක් අතර සන්නිවේදනය සඳහා භාවිතා කරන interface එකකි. එය එක් පද්ධතියක සේවාවන් අනෙකුත් පද්ධතිවලට භාවිතා කිරීමට අවසර දෙයි.

REST APIs වැනි මාර්ගයන් HTTP methods (GET, POST, PUT, DELETE) භාවිතා කර දත්ත හුවමාරු කරයි. සාමාන්‍යයෙන් JSON format භාවිතා වේ.

API design හොඳ වීමට clear endpoints, authentication, error handling සහ documentation වැදගත් වේ. Secure APIs සඳහා tokens සහ access control ක්‍රියාත්මක කළ යුතුය.

අමතර විස්තර (විස්තීරණය):
- ප්‍රධාන කරුණු: API ඉගෙනීමට concept -> pseudocode/plan -> implementation -> testing යන පියවර අනුගමනය කරන්න.
- ප්‍රායෝගික භාවිතය: කුඩා mini-project එකක් හරහා feature-by-feature build කිරීමෙන් theory දැනුම ක්‍රියාකාරී කුසලතාව බවට පත් වේ.
- සම්බන්ධ උපමාතෘකා: code quality, naming conventions, version control, documentation, සහ refactoring.
- හොඳ පුරුදු: commit discipline, automated tests, සහ code review පුරුදු කර ගත්තොත් errors අඩු කර maintainability වැඩි කරගත හැක.
