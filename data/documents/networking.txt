මාතෘකාව: Networking

Networking යනු පරිගණක හා උපකරණ එකිනෙකට සම්බන්ධ කර දත්ත හුවමාරු කිරීමයි. එය resource sharing සහ communication සඳහා මූලික තාක්ෂණයකි.

LAN (Local Area Network) කුඩා භූගෝලීය ප්‍රදේශයක භාවිතා වන අතර WAN (Wide Area Network) විශාල ප්‍රදේශ ආවරණය කරයි. Office සහ campus වැනි ස්ථානවල LAN බහුලව භාවිතා වේ.

Protocols භාවිතයෙන් devices අතර සම්මත සන්නිවේදනයක් පවත්වා ගනී. නිවැරදි network configuration මඟින් performance සහ reliability වැඩි කරගත හැක.

අමතර විස්තර (විස්තීරණය):
- ප්‍රධාන කරුණු: Networking සඳහා architecture components, data flow, සහ reliability considerations එකට බලන විට system understanding වැඩි වේ.
- ප්‍රායෝගික භාවිතය: setup -> monitor -> backup -> recovery යන workflow එකක් අනුගමනය කිරීමෙන් stable operations පවත්වා ගත හැක.
- සම්බන්ධ උපමාතෘකා: configuration management, observability, access control, සහ patch/update strategy.
- හොඳ පුරුදු: documentation update, periodic health checks, සහ security hardening steps ක්‍රමානුකූලව යෙදීම.
