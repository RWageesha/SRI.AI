මාතෘකාව: Encryption

Encryption යනු දත්ත සංකේතගත කර අනවසර පුද්ගලයින්ට අර්ථය නොපෙනෙන ලෙස ආරක්ෂා කිරීමේ ක්‍රමයකි. Data confidentiality සඳහා මෙය ප්‍රධාන තාක්ෂණයකි.

Plain text එක encrypted text (cipher text) බවට පරිවර්තනය කරයි. නිවැරදි key එක නොමැතිව මුල් දත්ත නැවත කියවීම අපහසු වේ.

Encryption internet communication, banking, messaging සහ file protection වැනි ක්ෂේත්‍රවල බහුලව භාවිතා වේ. දත්ත ආරක්ෂාව වැඩි කිරීමට key management නිවැරදිව කළ යුතුය.

අමතර විස්තර (විස්තීරණය):
- ප්‍රධාන කරුණු: Encryption සඳහා architecture components, data flow, සහ reliability considerations එකට බලන විට system understanding වැඩි වේ.
- ප්‍රායෝගික භාවිතය: setup -> monitor -> backup -> recovery යන workflow එකක් අනුගමනය කිරීමෙන් stable operations පවත්වා ගත හැක.
- සම්බන්ධ උපමාතෘකා: configuration management, observability, access control, සහ patch/update strategy.
- හොඳ පුරුදු: documentation update, periodic health checks, සහ security hardening steps ක්‍රමානුකූලව යෙදීම.
