මාතෘකාව: IP Address

IP Address යනු network එකක එක් device එකක් අනෙකාගෙන් වෙන්කර හඳුනාගැනීමට භාවිතා කරන අංකීය ලිපිනයකි. Data packets නිවැරදි destination එකට යොමු වීමට මෙය උපකාරී වේ.

IPv4 සහ IPv6 යන වර්ග දෙක ප්‍රධාන වේ. IPv4 ලිපිනයන් සීමිත වන අතර IPv6 වැඩි ලිපින අවකාශයක් ලබා දේ.

Network troubleshooting කිරීමේදී IP, subnet, gateway වැනි සංකල්ප වැදගත් වේ. IP තේරුම් ගැනීම network administration සඳහා මූලික දැනුමකි.

අමතර විස්තර (විස්තීරණය):
- ප්‍රධාන කරුණු: IP Address සඳහා architecture components, data flow, සහ reliability considerations එකට බලන විට system understanding වැඩි වේ.
- ප්‍රායෝගික භාවිතය: setup -> monitor -> backup -> recovery යන workflow එකක් අනුගමනය කිරීමෙන් stable operations පවත්වා ගත හැක.
- සම්බන්ධ උපමාතෘකා: configuration management, observability, access control, සහ patch/update strategy.
- හොඳ පුරුදු: documentation update, periodic health checks, සහ security hardening steps ක්‍රමානුකූලව යෙදීම.
