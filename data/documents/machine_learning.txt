මාතෘකාව: Machine Learning

Machine Learning (ML) යනු දත්ත මත පදනම්ව patterns ඉගෙනගෙන පුරෝකථන හෝ තීරණ ලබා දෙන AI ශාඛාවකි. Explicit rules සියල්ල ලියන්නේ නැතිව model එක දත්තෙන් ඉගෙන ගනී.

Supervised, unsupervised සහ reinforcement learning වැනි ප්‍රධාන ක්‍රම ඇත. Training data quality එක model performance එකට විශාල බලපෑමක් දරයි.

Model evaluation සඳහා accuracy, precision, recall වැනි metrics භාවිතා කරයි. Overfitting සහ data bias වැනි ගැටලු පාලනය කිරීම ML development හි වැදගත් කොටසකි.

අමතර විස්තර (විස්තීරණය):
- ප්‍රධාන කරුණු: Machine Learning සම්බන්ධ වැඩ වලදී data representation, accuracy, සහ consistency පදනම් කරගෙන සැලසුම් කිරීම වැදගත්ය.
- ප්‍රායෝගික භාවිතය: sample datasets/queries/examples භාවිතා කර output verify කිරීමෙන් concept correctness ඉක්මනින් තහවුරු කළ හැක.
- සම්බන්ධ උපමාතෘකා: indexing/search, data validation, scalability, සහ performance tuning.
- හොඳ පුරුදු: input constraints, edge cases, සහ failure handling කලින් සැලසුම් කිරීමෙන් production issues අඩු වේ.
