මාතෘකාව: ROM

ROM (Read Only Memory) යනු ස්ථිර මතකයකි. පද්ධතිය ආරම්භ වීමට අවශ්‍ය මූලික උපදෙස් මෙහි ගබඩා කර ඇත.

RAM මෙන් නොව ROM හි දත්ත විදුලිය නැති වූ පසුවද අහිමි නොවේ. එබැවින් firmware වැනි මූලික code ROM තුළ තබා ගනී.

Boot process එකේදී පද්ධතිය අවශ්‍ය ආරම්භක උපදෙස් ROM වෙතින් ලබා ගනී. මෙය system stability සඳහා වැදගත් කොටසකි.

අමතර විස්තර (විස්තීරණය):
- ප්‍රධාන කරුණු: ROM සඳහා architecture components, data flow, සහ reliability considerations එකට බලන විට system understanding වැඩි වේ.
- ප්‍රායෝගික භාවිතය: setup -> monitor -> backup -> recovery යන workflow එකක් අනුගමනය කිරීමෙන් stable operations පවත්වා ගත හැක.
- සම්බන්ධ උපමාතෘකා: configuration management, observability, access control, සහ patch/update strategy.
- හොඳ පුරුදු: documentation update, periodic health checks, සහ security hardening steps ක්‍රමානුකූලව යෙදීම.
