මාතෘකාව: පරිගණකය

පරිගණකය යනු දත්ත ලබාගෙන ඒවා සැකසීමෙන් තොරතුරු නිපදවන ඉලෙක්ට්‍රොනික යන්ත්‍රයකි. පරිගණකයක මූලික කාර්යයන් ලෙස input, processing, output සහ storage සලකයි.

Input යනු යතුරුපුවරුව, mouse, scanner වැනි උපකරණ හරහා දත්ත ලබා දීමයි. Processing යනු CPU මඟින් එම දත්ත සැකසීමයි.

Output ලෙස monitor, printer හෝ speaker හරහා ප්‍රතිඵල ලැබේ. Storage ලෙස hard disk, SSD, cloud වැනි මාධ්‍ය භාවිතා කර දත්ත තබා ගනී.

අමතර විස්තර (විස්තීරණය):
- ප්‍රධාන කරුණු: පරිගණකය සඳහා architecture components, data flow, සහ reliability considerations එකට බලන විට system understanding වැඩි වේ.
- ප්‍රායෝගික භාවිතය: setup -> monitor -> backup -> recovery යන workflow එකක් අනුගමනය කිරීමෙන් stable operations පවත්වා ගත හැක.
- සම්බන්ධ උපමාතෘකා: configuration management, observability, access control, සහ patch/update strategy.
- හොඳ පුරුදු: documentation update, periodic health checks, සහ security hardening steps ක්‍රමානුකූලව යෙදීම.
